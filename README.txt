Dans ce dépot Git vous trouverez 2 scripts : 

Un script bash qui peu s'executer directement dans le terminal
Un script python qui peu être appeler grâce a Python



Les 2 scripts permettent de faire la même chose:

Installer sur une nouvelle machine tout l'environnement necessaire pour travailler, VsCode, Discord, Git, Python3. Et retourne dans un fichier log.txt toutes les opérations effectuées.
